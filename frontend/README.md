
# Frontend - Mini Verifiable Credential Wallet

React application for managing verifiable credentials.


## Tech Stack

- React 19 - UI framework
- Vite - Build tool and dev server
- React Router 7 - Client-side routing
- TailwindCSS - Styling
- Axios - HTTP client


## Prerequisites

- Node.js
- npm


## Run Locally

Go to the frontend

```bash
  cd frontend
```

Install dependencies

```bash
  npm install
```

Environment Variables

```bash
  cp .env.example .env
```

To run this project, you will need to configure below environment variable.

```bash
    VITE_BACKEND_URL=your-backend-url-with-port
```

Start the server

```bash
  npm run dev
```

Frontend runs at: [http://localhost:5173](http://localhost:5173)
